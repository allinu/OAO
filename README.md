线上地址为：[liona.fun](https://liona.fun)

## 使用

如果你想使用本项目，请按照以下步骤

1. 克隆本项目
   ```shell
   git clone https://github.com/allinu/OAO.git
   ```
2. 切换到项目所在路径，并安装依赖
   ```shell
   cd OAO
   npm install || yarn
   ```
3. 启动服务

   ```shell
   npm run dev || yarn dev
   ```
